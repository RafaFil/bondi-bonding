# Bondi Bonding

Bondi bonding is a social app where you can find travel companions

## How to run it

- `git clone https://github.com/RafaFil/bondi-bonding`
- `cd bondi-bonding`
- `npm i`
- `ng serve -o`

## Host link
- https://bondi-bonding.netlify.app/
